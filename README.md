---
## **Commands**

- **.claim** - Claim the stand.
- **.s [name]** - Kill a player with stomp.
- **.k [name]** - Kill a player without stomp.
- **.r add [name]** - Add a player to Ragebot.
- **.r remove [name]** - Remove a player from Ragebot.
- **.r [name]** - Toggle a player in Ragebot (add if not there, remove if already added).
- **.mode 1** - Set teleport mode to normal.
- **.mode 2** - Set teleport mode to flick.
- **.aura on** - Enable aura.
- **.aura off** - Disable aura.
- **.aura 1** - Set aura mode to 1 (no stomp).
- **.aura 2** - Set aura mode to 2 (with stomp).
- **.aura add [name]** - Add a player to the aura-protected list.
- **.aura remove [name]** - Remove a player from the aura-protected list.
- **.aura range [number]** - Set the aura range.
- **.w add [name]** - Add a player to the whitelist.
- **.w remove [name]** - Remove a player from the whitelist.
- **.b [name]** - Bring a player to you.
- **.allow [name]** - Allow a player to use commands(Stomp, Bring and Kill).
- **.unallow [name]** - Remove a player from the allowed list.
